Glosario Alfabético

API
Interface de Programación de Aplicaciones, un conjunto de funciones y procedimientos que permiten el acceso a características o datos de un servicio o sistema operativo.
CSS
Hojas de Estilo en Cascada, un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML.
HTML
Lenguaje de Marcado de Hipertexto, el lenguaje estándar para crear páginas web.
JavaScript
Lenguaje de programación utilizado principalmente para el desarrollo web, permitiendo la creación de páginas web interactivas.
UX
Experiencia del Usuario, la percepción y respuesta de una persona resultante del uso o anticipación de uso de un producto, sistema o servicio.
